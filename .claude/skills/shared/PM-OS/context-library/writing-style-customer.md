# Customer-Facing Writing Style

Use for product marketing, help docs, customer communications.

## Characteristics

**Tone:** Friendly, helpful, clear
**Audience:** Customers and prospects
**Goal:** Make customers successful and happy

## Rules

### 1. Use Simple Language
- Avoid: "Leverage our robust API infrastructure"
- Better: "Connect your tools in minutes"

- Avoid: "Utilize the optimization functionality"
- Better: "Save time with automation"

### 2. Focus on Benefits, Not Features
- Avoid: "Advanced filtering with 20+ parameters"
- Better: "Find exactly what you need in seconds"

- Avoid: "Real-time synchronization"
- Better: "Your team always sees the latest updates"

### 3. Be Human
- Use "you" and "we"
- Write like you talk
- Show personality (within brand)
- Be warm but professional

### 4. Anticipate Questions
- Why should I care?
- How does this help me?
- What do I do next?
- What if something goes wrong?

### 5. Make It Scannable
- Short paragraphs (2-3 sentences)
- Clear headings
- Bullet points
- Bold key points
- Add visuals

## Example

**Bad:**
```
Our platform utilizes advanced machine learning algorithms to optimize
workflow efficiency, enabling enterprises to achieve unprecedented levels
of productivity through intelligent task prioritization and resource
allocation mechanisms.
```

**Good:**
```
## Get More Done with Smart Task Management

We help you focus on what matters most. [Your Product] automatically
prioritizes your tasks based on deadlines, importance, and your team's workload.

**Here's how it works:**
1. Add your tasks (or import from your tools)
2. We sort them by what needs your attention now
3. Focus on today's priorities, we'll handle tomorrow

**The result?** Teams finish [X]% more projects on time.

Ready to try it? Start your free trial ->
```

## Content Types

### Product Announcements
```
## [Feature Name] is Here!

**What it does:** [Clear benefit in one line]

**Why we built it:** [Customer problem it solves]

**How to use it:**
1. [Step 1]
2. [Step 2]
3. [Step 3]

**What customers are saying:**
"[Quote from beta customer]" - [Name], [Company]

Try it now: [Link]
Questions? We're here to help: [Support link]
```

### Help Documentation
```
## How to [Accomplish Task]

**Time needed:** [X minutes]
**Skill level:** [Beginner/Intermediate/Advanced]

[Brief overview of what this accomplishes]

**Steps:**
1. [Step 1 with screenshot]
2. [Step 2 with screenshot]
3. [Step 3 with screenshot]

**Troubleshooting:**
Problem: [Common issue]
Solution: [How to fix]

**Video walkthrough:** [Link]

**Still stuck?** Contact support ->
```

### Error Messages
```
Avoid: "Error 403: Insufficient permissions"

Better: "You don't have access to this file"
   [Request access] [Go back]

---

Avoid: "Network timeout exception"

Better: "Connection lost. Trying to reconnect..."
   [Having trouble?]
```

## When to Use
- Product marketing pages
- Blog posts
- Help documentation
- Email campaigns
- In-app messaging
- Social media
